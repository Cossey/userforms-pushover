# UserForms Pushover
Enables Pushover.net Support in UserForms module.

## Requirements
The SilverStripe UserForms Module.

## Installation
```sh
$ composer require stewartcossey/userforms-pushover
```
You will need to run `dev/build?flush=all` after installing.
